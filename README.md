# FBI — Forbes Bureau of Intelligence

Personal HQ for agents and automation tools. Each folder is a self-contained agent or experiment.

## Agents

| Folder | What it does |
|---|---|
| [`ai-signal`](ai-signal/) | On-demand AI/ML newsletter digest from tagged Gmail emails |
| [`aws-assistant`](aws-assistant/) | AWS expert assistant for ML engineers — live API-grounded answers |
| [`the-scientist-advice`](the-scientist-advice/) | Research-backed answers for data/ML practitioners, with paper saving |
| [`master-chef`](master-chef/) | Personal chef that plans weekly meals, maintains a recipe wiki and generates shopping lists — runs entirely inside Claude Code, no API key needed |

Each folder has its own `README.md` with setup instructions and a `SKILL.md` with the agent definition (or `CLAUDE.md` for wiki-pattern agents).

## Why FBI?

Because "Forbes Bureau of Intelligence" sounds more impressive than "stuff I built," and frankly, it's more accurate.
