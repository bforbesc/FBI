# Data & ML Research Workspace

Research notes and paper summaries for data engineers, ML engineers, and data scientists.

## Structure

```
the-scientist-advice/
├── config.md            ← edit here to change scope, sources, and tone
├── research-log.md      ← chronological log of all saved papers
└── papers/              ← all paper summaries, flat (one file per paper)
```

## How to add papers

In any session, say **"save that paper"** and the assistant will:
1. Create a plain-language summary as `papers/[paper-name].md`
2. Append an entry to `research-log.md`

## How to customize

Open `config.md` to:
- Change who the assistant serves
- Add or remove research sources
- Adjust language and tone rules
